## Scaffold commands:

-  Start new project
	```bash
	npx truffle init project
	```
    
- Scaffold a contract
	```bash
	truffle create contract YourContractName # scaffold a contract
	```
    
- Scaffold a test
	```bash
	truffle create test YourTestName         # scaffold a test
	```
	
	
- Compile
	```bash
	npx truffle compile
	```
